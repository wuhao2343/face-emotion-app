本地 API

所有端点以 /api/ 开头,CORS 已全开。Swagger 文档:/docs

GET /api/health

服务状态、运行时 provider、模型加载情况。

POST /api/detect/image

上传图片,返回带框 + 情绪标签的 JPEG。响应头 X-Face-Count 包含人脸数。

POST /api/detect/frame

上传图片,返回 JSON(供其他程序调用):

{
  "count": 1,
  "inference_ms": 35.2,
  "device": "CPUExecutionProvider",
  "detections": [
    {
      "bbox": {"x1": 120, "y1": 80, "x2": 280, "y2": 320},
      "emotion": "happy",
      "score": 0.92,
      "all_scores": {"angry": 0.01, "disgust": 0.0, "fear": 0.01,
                     "happy": 0.92, "sad": 0.02, "surprise": 0.03,
                     "neutral": 0.01}
    }
  ]
}

POST /api/detect/video

上传视频,返回 {"task_id": "..."}。轮询 /api/detect/video/{task_id}/status 拿进度,完成后从 /api/detect/video/{task_id}/download 下载 MP4。

GET /api/camera/list

列出可用摄像头。

GET /api/camera/{id}/stream

MJPEG 流,直接在 <img> 标签中使用。

命令行示例

# JSON 检测
curl -F "file=@face.jpg" http://localhost:8000/api/detect/frame | jq

# 图片打码返回
curl -F "file=@face.jpg" http://localhost:8000/api/detect/image -o result.jpg

# 视频处理
curl -F "file=@clip.mp4" http://localhost:8000/api/detect/video
# {"task_id":"abc123","status":"queued"}
curl http://localhost:8000/api/detect/video/abc123/status
curl -o out.mp4 http://localhost:8000/api/detect/video/abc123/download
